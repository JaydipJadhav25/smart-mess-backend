# Smart Mess System – Backend

The Smart Mess System Backend provides APIs for student attendance, fee management, meal menu, authentication, feedback, and admin operations.

---

## Features
- Student Authentication (JWT)
- Attendance Tracking
- Fee Payment Management
- Weekly Meal Menu
- Student Feedback System
- Admin APIs
- MongoDB Database (Mongoose)
- Optional OpenAI API Integration

---

## Installation

### 1. Clone the Repository
```bash
git clone https://github.com/JaydipJadhav25/smart-mess-backend.git
cd smart-mess-backend
```


### 2. Install Dependencies
```
npm install
```
### Running the Project
### Development
```
npm run de

```
